## Networking

- **LAN**: 172.16.1.0/24 — all physical nodes
- **Headscale**: 100.64.0.0/24 — VPN overlay across all locations
- **DNS**: Pi-hole (ad blocking) → CoreDNS (service discovery) → Cloudflare (public)
- **Ingress**: Traefik on K3s with automatic TLS via Let's Encrypt
- **Split DNS**: `*.staging.kubelab.live` resolves to K3s via VPN, production uses Cloudflare

## IaC Pipeline

| Layer | Tool | Source of truth |
|-------|------|-----------------|
| DNS | Terraform | `infra/terraform/dns/` |
| Node provisioning | Ansible | `infra/ansible/` |
| K8s manifests | Kustomize | `infra/k8s/base/` + overlays |
| Secrets | SOPS + toolkit | `infra/config/values/*.enc.yaml` |
| Config values | YAML | `infra/config/values/common.yaml` |

## Auth Flow

All HTTP services go through Traefik → Authelia forward-auth middleware. Authelia provides:

- OIDC provider for Grafana, MinIO, and future services
- Two-factor authentication (TOTP)
- Access control policies per subdomain
- Session persistence via Redis
